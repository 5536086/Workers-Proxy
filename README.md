# 装模做样的写个简介

本库中除“README.md”之外都是镜像网站的部署脚本，在[Cloudflare Workers](https://workers.cloudflare.com/)中使用
直接复制粘贴，无需其他额外操作，点击“保存并部署”后生效

目前本库中有：
1. GitHub的镜像脚本
2. Google的镜像脚本
3. Google-scholar的镜像脚本
4. Nyaa的镜像脚本
5. Nytimes的镜像脚本
6. Wikipedia-en的镜像脚本
7. Wikipedia-zh的镜像脚本
8. YouTube的镜像脚本
