# 装模做样的写个简介

本库中除“README.md”之外都是镜像网站的部署脚本，在[Cloudflare Workers](https://workers.cloudflare.com/)中使用

直接复制粘贴相关脚本，无需其他额外操作，点击“保存并部署”后生效

目前本库中有：
01. GitHub的镜像脚本
02. Google的镜像脚本
03. Google-scholar的镜像脚本
04. Nyaa的镜像脚本
05. Nytimes的镜像脚本
06. Pixiv的镜像脚本
07. Sukebei的镜像脚本
08. Wikipedia-en的镜像脚本
09. Wikipedia-zh的镜像脚本
10. YouTube的镜像脚本
11. 灵梦御所的镜像脚本
