# 装模做样的写个简介

本库中除“README.md”之外都是镜像网站的部署脚本，在[Cloudflare Workers](https://workers.cloudflare.com/)中使用

直接复制粘贴相关脚本，无需其他额外操作，点击“保存并部署”后生效

目前本库中有：
01. Figma的镜像脚本
02. GitHub的镜像脚本
03. Google的镜像脚本
04. Google-scholar的镜像脚本
05. Nyaa的镜像脚本
06. Nytimes的镜像脚本
07. Pixiv的镜像脚本
08. Sukebei的镜像脚本本
09. Wikipedia-en的镜像脚
10. Wikipedia-zh的镜像脚本
11. YouTube的镜像脚本
12. 灵梦御所的镜像脚本
